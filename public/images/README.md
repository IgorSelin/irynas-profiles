# Зображення для сайту

Додайте наступні зображення у цю папку:

## Обов'язкові зображення:

1. **hero-bg.jpg** (1920x1080 або більше)
   - Фон для Hero секції
   - Рекомендовано: фото Львова, історичний центр

2. **guide-photo.jpg** (800x800 або більше, квадратне)
   - Фото екскурсовода для секції "Про мене"
   - Рекомендовано: професійне фото

3. **tour-historical.jpg** (1200x800 або більше)
   - Фото для екскурсії "Історичний центр Львова"
   - Рекомендовано: Площа Ринок, історичні будівлі

4. **tour-legends.jpg** (1200x800 або більше)
   - Фото для екскурсії "Львівські легенди та міфи"
   - Рекомендовано: таємничі місця, старовинні дворики

5. **tour-coffee.jpg** (1200x800 або більше)
   - Фото для екскурсії "Кавовий Львів"
   - Рекомендовано: кав'ярні, кава, атмосфера

6. **tour-architecture.jpg** (1200x800 або більше)
   - Фото для екскурсії "Архітектурні перлини"
   - Рекомендовано: красиві будівлі, архітектурні деталі

## Формати:
- JPG або PNG
- Оптимізуйте зображення для веб-використання
- Використовуйте якісні фото для кращого враження

## Оптимізація:
Рекомендується використовувати інструменти для оптимізації зображень перед завантаженням (наприклад, TinyPNG, ImageOptim).

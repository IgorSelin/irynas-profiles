# Налаштування інтеграції з Google Maps

Для того, щоб фотографії з вашого профілю Google Maps автоматично з'являлися в галереї, виконайте наступні кроки:

## 1. Отримання API Ключа

1. Перейдіть до [Google Cloud Console](https://console.cloud.google.com/).
2. Створіть новий проект (або виберіть існуючий).
3. У розділі **APIs & Services > Library** знайдіть та увімкніть **Places API**.
4. Перейдіть до **APIs & Services > Credentials**.
5. Натисніть **Create Credentials > API key**.
6. Скопіюйте створений ключ.

## 2. Додавання ключа до проекту

Створіть файл `.env.local` у кореневій папці проекту (якщо його ще немає) та додайте туди ваш ключ:

```env
GOOGLE_MAPS_API_KEY=ваш_отриманий_ключ
```

## 3. Налаштування Place ID

У файлі `app/api/photos/route.ts` встановлено тимчасовий Place ID. Для вашого бізнесу "Екскурсії Львовом з Іриною Красіцькою" він зазвичай визначається автоматично, але якщо ви хочете переконатися:

1. Знайдіть свій бізнес на Google Maps.
2. Скопіюйте частину URL або скористайтеся [Place ID Finder](https://developers.google.com/maps/documentation/javascript/examples/full/place-id-finder).
3. Оновіть змінну `PLACE_ID` у `app/api/photos/route.ts`.

## Як це працює

- Сайт автоматично підтягує останні фотографії, додані до вашого профілю Google Maps.
- Фотографії кешуються для швидкого завантаження.
- Якщо API ключ не налаштований, галерея показуватиме лише статичні фотографії з папки `public/images`.

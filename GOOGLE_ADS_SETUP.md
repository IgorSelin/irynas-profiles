# Налаштування Google Ads та SEO для таргетованої реклами

## Частина 1: Google Ads (Платна реклама)

### Крок 1: Створення Google Ads акаунту

1. Перейдіть на [Google Ads](https://ads.google.com/)
2. Створіть акаунт або увійдіть у існуючий
3. Виберіть тип кампанії: **Пошукова мережа** (Search Network)

### Крок 2: Налаштування кампанії

#### Основні налаштування:

- **Назва кампанії:** "Екскурсії Львів"
- **Тип кампанії:** Пошук
- **Мета:** Веб-сайт
- **Денний бюджет:** 65-70 грн/день (місячний бюджет: 2000 грн)

#### Таргетинг (Дуже важливо для правильної аудиторії):

**Географічний таргетинг:**

- Львів, Україна (основна локація)
- Львівська область
- Можна додати сусідні області (Івано-Франківська, Тернопільська)

**Мови:**

- Українська
- Польська

**Демографічний таргетинг:**

- Вік: 25-65 років
- Стать: Всі
- Інтереси: Подорожі, Туризм, Культура, Історія

**Пристрої:**

- Мобільні пристрої (важливо, оскільки багато туристів шукають на телефоні)
- Комп'ютери
- Планшети

### Крок 3: Ключові слова (Keywords)

**Основні ключові слова (високий пріоритет):**

- екскурсовод Львів
- екскурсії Львів
- тури Львів
- гід Львів
- екскурсії по Львову
- львівський екскурсовод
- індивідуальні екскурсії Львів
- групові тури Львів
- екскурсії Львів українською
- екскурсії Львів польською
- середньовічний Львів екскурсія
- підземелля Львова
- оперний театр Львів екскурсія
- Личаківське кладовище екскурсія

**Довгі ключові слова (Long-tail):**

- де знайти екскурсовода у Львові
- професійний гід Львів
- екскурсії по історичному центру Львова
- тури по Львову з гідом
- екскурсії Львів ціна

**Негативні ключові слова (щоб не показувати рекламу):**

- безкоштовно
- дешево
- самотужки
- самостійно

### Крок 4: Створення оголошень (Ads)

**Заголовок 1:** Екскурсовод Львів | Професійні тури
**Заголовок 2:** Індивідуальні та групові екскурсії
**Заголовок 3:** Українською та польською мовами

**Опис 1:** Професійний екскурсовод у Львові. 8 унікальних маршрутів. Середньовічний Львів, підземелля, Оперний театр. 200 грн/ос.

**Опис 2:** Досвідчений гід з 5+ роками стажу. Екскурсії по історичному центру, австрійському Львову, Личаківському кладовищу. Замовити: +380975383348

**Шлях відображення:**

- www.yourdomain.com/екскурсії
- www.yourdomain.com/тури

### Крок 5: Розширення оголошень

**Розширення сайту:**

- Головна
- Екскурсії
- Відгуки
- Контакти

**Розширення виклику:**

- +380 97 538 33 48

**Розширення структурованих фрагментів:**

- Відгуки (з Firebase)
- Ціни (200 грн/ос)
- Мови (Українська, Польська)

### Крок 6: Конверсії

Налаштуйте відстеження конверсій:

- Дзвінки (кліки на телефон)
- Запити через форму (якщо додасте)
- Перегляди сторінки екскурсій

## Частина 2: Google Search Console (Безкоштовний SEO)

### Крок 1: Підключення сайту

1. Перейдіть на [Google Search Console](https://search.google.com/search-console)
2. Додайте ваш сайт
3. Виберіть метод верифікації:
   - HTML тег (додайте у layout.tsx)
   - HTML файл
   - Google Analytics
   - DNS запис

### Крок 2: Налаштування

**Sitemap:**

- Додайте ваш sitemap: `https://yourdomain.com/sitemap.xml`

**Міжнародне націлення:**

- Основна країна: Україна
- Мова: Українська
- Альтернативна мова: Польська

### Крок 3: Моніторинг

Перевіряйте:

- Індексовані сторінки
- Пошукові запити
- Кліки та покази
- Помилки індексації

## Частина 3: Google Analytics (Аналітика)

### Крок 1: Створення проєкту

1. Перейдіть на [Google Analytics](https://analytics.google.com/)
2. Створіть проєкт GA4
3. Отримайте Measurement ID (виглядає як `G-XXXXXXXXXX`)

### Крок 2: Додавання на сайт

1. Додайте `NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX` у `.env.local`
2. Компонент GoogleAnalytics вже додано у layout.tsx
3. Перезапустіть сервер

### Крок 3: Налаштування подій

Відстежуйте:

- Кліки на телефон
- Перегляди екскурсій
- Відправка відгуків
- Скрол до секцій

## Частина 4: Покращення SEO для органічного пошуку

### Оптимізація контенту

1. **Додайте більше тексту про екскурсії** - Google любить багатий контент
2. **Використовуйте ключові слова природно** у текстах
3. **Додайте FAQ секцію** з питаннями типу:
   - "Скільки коштує екскурсія по Львову?"
   - "Які мови доступні для екскурсій?"
   - "Як замовити екскурсію?"

### Локальний SEO

1. **Google Business Profile:**
   - Створіть профіль у Google Business
   - Додайте фото, відгуки, години роботи
   - Отримайте рейтинг

2. **Структуровані дані:**
   - Вже додано у layout.tsx
   - Можна додати LocalBusiness schema

### Зовнішні посилання

1. Реєстрація на туристичних порталах
2. Соціальні мережі (Facebook, Instagram)
3. Партнерські посилання з інших сайтів про Львів

## Частина 5: Бюджет та стратегія

### Рекомендований бюджет для початку:

- **Денний бюджет:** 60-70 грн (для місячного бюджету 2000 грн)
- **Місячний бюджет:** 2000 грн (оптимально для початку)
- **CPC (ціна за клік):** 5-15 грн (залежить від конкуренції)

### Стратегія для бюджету 2000 грн/місяць:

**Розподіл бюджету:**

- **Денний бюджет:** 65-70 грн (2000 грн / 30 днів ≈ 67 грн/день)
- **Очікувані кліки:** 4-13 кліків на день (залежить від CPC)
- **Очікувані кліки на місяць:** 120-390 кліків

**Оптимізація для невеликого бюджету:**

1. **Фокус на найефективніших ключових словах:**
   - "екскурсовод Львів" (високий пріоритет)
   - "екскурсії Львів" (високий пріоритет)
   - "тури Львів" (середній пріоритет)
   - "гід Львів" (середній пріоритет)

2. **Використовуйте точкове співпадіння (Exact Match):**
   - Зменшує витрати на нерелевантні запити
   - Підвищує якість трафіку

3. **Обмежте покази:**
   - Тільки пошукова мережа (вимкніть Display Network на початку)
   - Тільки мобільні пристрої (якщо більшість клієнтів шукають на телефоні)

4. **Максимізуйте конверсії:**
   - Додайте розширення виклику (Call Extension)
   - Додайте розширення сайту
   - Оптимізуйте сторінку приземлення (landing page)

5. **Моніторинг та оптимізація:**
   - Перевіряйте результати щодня
   - Видаляйте нерезультативні ключові слова
   - Збільшуйте ставки на успішні ключові слова

### Оптимізація для бюджету 2000 грн/місяць:

- **Починайте з денного бюджету 60-70 грн**
- **Фокус на 5-10 найважливіших ключових словах**
- Аналізуйте які ключові слова дають конверсії (дзвінки)
- Збільшуйте ставки на успішні ключові слова (якщо бюджет дозволяє)
- Видаляйте нерезультативні ключові слова негайно
- Використовуйте точкове співпадіння для економії бюджету
- Вимкніть Display Network на початку (тільки пошук)

## Частина 6: Таргетування аудиторії

### Ідеальна аудиторія:

- **Локація:** Львів та Львівська область
- **Вік:** 25-65 років
- **Інтереси:** Подорожі, Туризм, Культура, Історія, Архітектура
- **Поведінка:** Шукають тури, екскурсії, подорожі
- **Пристрої:** Мобільні (60-70% трафіку)

### Ремаркетинг:

- Створіть аудиторію відвідувачів сайту
- Показуйте рекламу тим, хто вже відвідував сайт
- Це збільшить конверсії

## Додаткові поради

1. **A/B тестування:** Створюйте кілька варіантів оголошень
2. **Адаптивність:** Переконайтеся, що сайт добре виглядає на мобільних
3. **Швидкість:** Оптимізуйте швидкість завантаження (Google це оцінює)
4. **Відгуки:** Збирайте та відображайте відгуки (вже реалізовано)
5. **Оновлення контенту:** Регулярно оновлюйте інформацію про екскурсії
